🚀 Rota do Código

Aplicação web desenvolvida com foco na apresentação institucional do projeto Rota do Código, uma iniciativa voltada à educação tecnológica e introdução à programação.

📄 Descrição

O projeto consiste em uma interface web responsiva que apresenta informações sobre a proposta, objetivos e equipe do Rota do Código. A aplicação foi construída com foco em organização de componentes, reutilização de código e boas práticas de desenvolvimento front-end.
